# 🌸 Iris Flower Classification using AdaBoost

A Machine Learning project that predicts the species of an Iris flower using the AdaBoost Classification algorithm. The project includes data preprocessing, model training, evaluation, and a user-friendly Streamlit web application for real-time predictions.

---

## 📌 Project Overview

The Iris dataset is one of the most popular datasets in Machine Learning. This project uses the AdaBoost (Adaptive Boosting) algorithm to classify Iris flowers into three species:

* Iris-setosa
* Iris-versicolor
* Iris-virginica

Users can enter flower measurements through a Streamlit interface and instantly receive a prediction.

## 📂 Dataset Features

The model uses the following input features:

| Feature      | Description          |
| ------------ | -------------------- |
| Sepal Length | Length of sepal (cm) |
| Sepal Width  | Width of sepal (cm)  |
| Petal Length | Length of petal (cm) |
| Petal Width  | Width of petal (cm)  |

### Target Variable

* Species

## 🎯 Model Used

### AdaBoost Classifier

AdaBoost (Adaptive Boosting) is an ensemble learning algorithm that combines multiple weak learners to create a strong predictive model.

**Advantages:**

* High accuracy
* Reduces overfitting
* Works well on classification tasks
* Easy to implement
